# Hi, I'm tai

I build agent-first software, learning tools, and developer tooling.

I am currently focused on software that humans install and supervise, but agents can operate directly through strong CLI surfaces, structured output, and safe automation workflows.

## Current Projects

- [ankicli](https://github.com/Takhoffman/ankicli) - agent-first Anki CLI for studying, card authoring, deck management, media workflows, and automation.
- [winmote](https://github.com/Takhoffman/winmote) - remote control and automation tooling for Windows workflows.
- [agent-scripts](https://github.com/Takhoffman/agent-scripts) - reusable automation and maintainer scripts for agent-driven workflows.

## Current Direction

Right now I am especially interested in a new era of software where:

- humans choose the tools, install them, and set the constraints
- agents operate the software directly
- the product surface is designed for both supervision and execution

That shows up in my work as structured CLIs, installable skills, automation primitives, and products that are easy to hand off to an agent harness without losing control.
